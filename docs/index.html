Welcome to the RustGL API!

The one-stop-shop guide to an advanced graphics library. The RustGL library includes features needed to generate feature-rich GUI applications, like end-user software, video games, etc.

Dependencies

Like all good software, RustGL has a slew of required dependencies. If you are new to Rust, these dependencies are already included with the RustGL library. Here is the current dependencies. These dependencies can be found in the Cargo.toml file within the root of the project.

Crate Usage Category
Window building software. Used to supply the application window.
Graphics backend used to render UI and models in the application window.
OBJ file loader. Used to load model files into vertex data for rendering.
Used for advanced mathematics. Matrix and Vector types as well as linear algebra.
Macro library used for quickly generating map types, such as HashMaps.
Used for type-casting and associated types.
Macro library used for quickly creating C-strings.
